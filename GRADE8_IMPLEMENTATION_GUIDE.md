## 🎓 Phạm Vi Bài Giảng (20 Chủ Đề)

### I. Tiếng Anh (8 chủ đề)
1. **Present Perfect Tense** - Hiện tại Hoàn Thành
   - Cấu trúc: have/has + V3
   - Từ khóa: already, yet, ever, never, since, for
   - Bài tập: 5 bài

2. **Reported Speech** - Lời Nói Gián Tiếp
   - Backshift thì: Present → Past
   - Thay đổi đại từ, trạng từ thời gian
   - Bài tập: 3 bài

3. **Comparative & Superlative** - So Sánh & Tuyệt Đối
   - Tính từ ngắn: -er/the -est
   - Tính từ dài: more/the most
   - Bất quy tắc: good/bad/far
   - Bài tập: 4 bài

4. **Conditional Sentences** - Câu Điều Kiện
   - Type 1, 2, 3
   - Unless, mixed conditionals
   - Bài tập: 3 bài

5. **Passive Voice** - Câu Bị Động
   - Tất cả thì: Present, Past, Future, Perfect
   - Cấu trúc: be + V3
   - Bài tập: 3 bài

6. **Phrasal Verbs** - Động Từ Cụm
   - Come, Put, Look, Get
   - Separable/Inseparable
   - Bài tập: 4 bài

7. **Reading Comprehension** - Đọc Hiểu
   - Tìm ý chính, chi tiết, suy luận
   - Kỹ năng skim/scan
   - Bài tập: 2 bài

8. **Paragraph Writing** - Viết Đoạn Văn
   - Descriptive, Narrative, Expository
   - Topic sentence, supporting, conclusion
   - Bài tập: Nhiều

### II. Toán Học (7 chủ đề)
1. **Linear Equations** - Phương Trình Bậc Nhất
   - Giải phương trình
   - Kiểm tra nghiệm
   - Bài tập: 5 bài

2. **Linear Inequalities** - Bất Phương Trình Bậc Nhất
   - Đảo dấu khi nhân/chia âm
   - Biểu diễn trên trục số
   - Bài tập: 3 bài

3. **Systems of Equations** - Hệ Phương Trình
   - Phương pháp thế, cộng
   - Hệ vô nghiệm, vô số nghiệm
   - Bài tập: 3 bài

4. **Polynomials & Factorization** - Đa Thức & Phân Tích
   - Đặt nhân tử chung
   - Hằng đẳng thức
   - Nhóm hạng tử
   - Bài tập: 4 bài

5. **Triangle Congruence** - Tam Giác Bằng Nhau
   - CCC, CGC, GCG
   - Cạnh-góc tương ứng
   - Bài tập: 3 bài

6. **Quadrilaterals** - Tứ Giác & Tính Chất
   - Hình thang, bình hành, chữ nhật, thoi, vuông
   - Tính chất, nhận biết
   - Bài tập: 4 bài

7. **Statistics & Probability** - Thống Kê & Xác Suất
   - Tần số, tần suất, trung bình, trung vị, mốt
   - Xác suất cơ bản
   - Bài tập: 4 bài

### III. Ngữ Văn (4 chủ đề)
1. **Narrative Poetry** - Thơ Tự Sự
   - Cấu trúc: Mở, phát triển, cao trào, kết
   - Phân tích nhân vật, chủ đề
   - Ví dụ: Truyện Kiều

2. **Lyric Poetry** - Thơ Ch抒情
   - Cảm xúc, tâm tư, suy tư
   - Hình ảnh trung tâm
   - Ví dụ: Vội vàng, Chiều tối

3. **Short Story** - Truyện Ngắn
   - Tình tiết, nhân vật, bối cảnh
   - Phân tích sâu sắc
   - Ví dụ: Gà trống, Số phận

4. **Essay Writing** - Viết Văn Chủ Đề
   - Mở bài, thân bài, kết bài
   - Persuasive, Expository, Narrative
   - Dài 500-1000 từ

### IV. Khoa Học Tự Nhiên (4 chủ đề)
1. **Atoms & Molecules** - Nguyên Tử & Phân Tử
   - Cấu trúc nguyên tử
   - Ký hiệu nguyên tố
   - Hoá trị

2. **Chemical Reactions** - Phản Ứng Hóa Học
   - Cân bằng phương trình
   - Phân loại phản ứng
   - Dấu hiệu nhận biết

3. **Electricity** - Điện Học
   - Công thức Ohm: U = I × R
   - Mạch nối tiếp & song song
   - Cường độ, hiệu điện thế, điện trở

4. **Waves & Light** - Sóng & Ánh Sáng
   - Bước sóng, tần số: v = λ × f
   - Phản xạ, khúc xạ
   - Thấu kính

### V. Lịch Sử & Địa Lý (5 chủ đề)
1. **World War II** - Thế Chiến II
   - Nguyên nhân, diễn biến, hậu quả
   - Ảnh hưởng đến Việt Nam

2. **Vietnamese History** - Lịch Sử Việt Nam
   - Các giai đoạn: Tiền sử, Bắc thuộc, Độc lập, Pháp thuộc
   - Các nhân vật lịch sử

3. **Geography of Vietnam** - Địa Lý Việt Nam
   - Vị trí, diện tích, địa hình
   - Khí hậu, dân cư, kinh tế

(Ghi chú: Lịch sử & Địa Lý được kết hợp vì đây là môn Lịch Sử & Địa Lý trong chương trình TP.HCM)

## 📂 Cấu Trúc Tệp

### `grade8Lessons.ts`
```typescript
interface Lesson {
  id: string;              // g8-eng-1, g8-math-1, etc.
  subject: string;         // english, math, literature, science, history_geography
  topic: string;           // present-perfect, linear-equations, etc.
  title: string;           // Tên chủ đề
  theory: string;          // Giải thích chi tiết
  examples: string[];      // Ví dụ cụ thể
  practicePoints: string[]; // Các điểm cần luyện tập
  category: string;        // grammar, vocabulary, reading, etc.
  difficulty: number;      // 1-10
}

GRADE8_LESSONS: Lesson[] = [20 lessons]
```

### `grade8Questions.ts`
```typescript
interface Question {
  id: string;                          // g8-eng-1, g8-math-1, etc.
  type: 'mcq' | 'text_input' | 'short-answer' | 'reading';
  category: string;                    // present-perfect, linear-equations, etc.
  prompt: string;                      // Câu hỏi
  options?: string[];                  // Các lựa chọn (MCQ)
  correctAnswer: string | string[];    // Đáp án đúng
  explanation: string;                 // Giải thích
  difficulty: number;                  // 1-10
  source: string;                      // Grade 8 HCMC Curriculum
  subject: string;                     // Môn học
  metadata?: object;                   // Thông tin bổ sung
}

GRADE8_QUESTIONS: Question[] = [65 questions]
```

```typescript

// Lấy câu MCQ
const mcqQuestions = GRADE8_QUESTIONS.filter(q => q.type === 'mcq');
```

### 4. Tạo Quiz

```typescript
// Chọn 10 câu ngẫu nhiên
const quizQuestions = GRADE8_QUESTIONS
  .sort(() => Math.random() - 0.5)
  .slice(0, 10);

// Chọn câu từ chủ đề cụ thể
const linearEquationsQuiz = GRADE8_QUESTIONS
```

### 5. Kiểm Tra Đáp Án

```typescript
function checkAnswer(question, userAnswer) {
  const correctAnswers = Array.isArray(question.correctAnswer)
    ? question.correctAnswer
    : [question.correctAnswer];
  
  const normalized = userAnswer.toLowerCase().trim();
  return correctAnswers.some(ans => ans.toLowerCase().trim() === normalized);
}
```
